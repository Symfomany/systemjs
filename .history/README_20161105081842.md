### Demo SystemJS & TypeScript for Angular 2


SystemJS can transpile into:
 
+ Babel
+ TypeScript
+ Traceur


SystemJS respect all module:

 + ES6 modules
 + AMD
 + CommonJS
...  global scripts in the browser and NodeJS.
